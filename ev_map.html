What are the Percentages of Electric Vehicles in each County?

Reflection:

The purpose of this graph is to show how the percentage of electric vehicles is changing over time for the state of Washington. I used the position of the map with county boundary lines to represent the counties in Washington. Then, the quantitative variable for electric vehicle percentage was encoded using color on top of the county. I chose the yellow - green - blue color range and binned it to better show larger values with darker colors. In addition, there is a user interation. The user can select between the year 2021, 2022, and 2023 to see how the precentages of EVs out of total vehicles are going up each year. Finally, to better effective display the data, I added a tooltip, to show the exact value for each county.
